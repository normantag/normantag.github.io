Android

使用google官方教程

将 https://android.googlesource.com/ 全部使用 https://aosp.tuna.tsinghua.edu.cn/ 代替即可。

获取repo工具

如果不能正常，可以使用清华大学的git repo镜像

在~/.bashrc里添加（repo的运行过程中会尝试访问官方的git源更新自己）

使用清华大学的初始化包更新源码

初始化包超过30G，包含了所有的.repo 目录，由于所有代码都是从隐藏的 .repo 目录中 checkout 出来的，所以只要 repo sync 即可（AOSP更新非常快，即使使用初始化包也可能会联网更新）

conda/Anaconda/miniconda

Anaconda 是一个用于科学计算的 Python 发行版，支持 Linux, Mac, Windows, 包含了众多流行的科学计算、数据分析的 Python 包。

Miniconda 是一个 Anaconda 的轻量级替代，默认只包含了 python 和 conda，但是可以通过 pip 和 conda 来安装所需要的包。
