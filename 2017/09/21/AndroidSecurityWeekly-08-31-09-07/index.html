  3. 来自SuSE Labs的João Moreira在2017年的BlackHat Asia 2017上分享的关于内核CFI实现的议题

    DROP THE ROP: Fine Grained Control-Flow Integrity for The Linux Kernel slide:
    PPT: O网页链接 white paper: O网页链接 视频：O网页链接

    这个被称为kCFI的原型演示一种防御内核ROP的CFI机制，粗粒度的CFI容易被绕过，kCFI是一个基于LLVM的静态计算CFGs的细粒度CFI实现，这个原型在函数原型聚类方面的优化使性能看起来不错 O网页链接 这个议题公开至今已经接近6个月，但离工程化依然非常遥远 O网页链接 使用LLVM的instrumentation导致必须每次必须编译两次分别对源代码和二进制进行分析，而原型的内核版本是3.19，另外kCFI需要额外的工具也是一个工程上目前没有解决的问题，相比PaX的RAP能不借助于额外工具的情况下覆盖4.5到4.12 O网页链接 kCFI的工程化之路还很漫长
